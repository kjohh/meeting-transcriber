Meeting Transcriber

待機
歡迎 第 1 步 / 共 4 步
👋
Meeting Transcriber 會同時錄你的麥克風與電腦播放的聲音，即時轉成逐字稿。

線上、實體、混合會議都能用。

接下來幾步帶你完成初始設定。
🔓
要錄製會議,Meeting Transcriber 需要兩個權限。分別點按鈕授權即可。
📺 螢幕錄製 尚未授權
擷取電腦播放的聲音 — 線上會議對方的聲音 / 影片聲音。不會錄你的畫面。
🎙 麥克風 尚未授權
擷取你的聲音(線上會議發言、實體會議錄音)。授權後會即時顯示波形讓你測試。
macOS 15 之後螢幕錄製每個月會跳一次確認對話框(藍色舉手圖示),按「允許」即可。
⚙
選擇轉錄方式
兩種之後都可以再切換，先選一個開始即可。
✅
設定完成

按工具列的 ● 開始 錄你的第一場會議。
之後想改設定或重看這份說明，點右上角的 ⚙ 即可。
設定
Groq API 金鑰
只有雲端模式需要。免費註冊金鑰:console.groq.com/keys
說明
忘了怎麼用?重新看一次引導。
本機模型
下載符合你會議語言的模型。檔案存在 ~/Library/Application Support/pywhispercpp/models/, 與 lazy-take-notes 共用同個資料夾。
自訂詞彙
會議常出現的專有名詞、品牌、縮寫、人名放這裡, Whisper 模型會優先辨識成這些詞,避免被聽成相似發音的字。
每行一個,以 # 開頭的行是註解。儲存後下一段轉錄就會生效。
